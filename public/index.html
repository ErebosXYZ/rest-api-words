REST API Words

Esta API te permite obtener palabras aleatorias.

Endpoints v1

  • GET /api/v1/words
    Devuelve una palabra aleatoria.
  • GET /api/v1/words?length=3
    Devuelve una palabra aleatoria de 3 letras (entre 3 y 13).

Ejemplo de respuesta v1

{
  "word": "mar"
}

Endpoints v2

  • GET /api/v2/words?length=5&lang=pt-br
    Devuelve una palabra aleatoria según los parámetros establecidos en "length" y "lang" en la query string.
